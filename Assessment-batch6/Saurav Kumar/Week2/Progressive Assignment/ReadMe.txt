Created a Restaurant Tracker monolithic application using J2EE. It enables customers to book tables and order food for home delivery.

On Welcome page customer can opt for sign in or sign up.

After filling the credentials, customer lands on home page which has a list of all the branches of the restaurant.

Customer can select any branch. After that it shows menu and number of tables available for booking.

Customer can opt for table booking or home delivery.

For table booking, customer needs to input number of tables to book and click on reserve button.

For home delivery, customer needs to input quantity of each item in the menu. Total price appears on the confirmation page and on clicking confirm button, the order gets placed.

Customer can see the status of order by clicking status button on Thankyou page after placing the order successfully.
